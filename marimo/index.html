绿藻球（Marimo Moss Ball）

水草 图鉴

绿藻球（Marimo moss ball）比较知名的产地有日本的阿寒湖，在日本绿藻球被视为幸运的象征，由于绿藻球是藻类团生生物，因此绿藻球没有严格的生命限制，如果饲养得当可以超过一百年。相传只要能妥善的照顾球藻，就能够愿望成真。因此在日本新人结婚的时候也会收到绿藻球，以祝福他们的爱情一直长久，另外绿藻球也经常在朋友间互相赠送。

中文名：绿藻球、藻球、海藻球
汉字学名：球藻、毡藻
拉丁学名：Aegagropila linnaei
英文名：Marimo, Moss Ball
分类学：植物界 → 绿藻门 → 石莼纲 → 刚毛藻目 → 刚毛藻科 → 球藻属
原产地：日本阿寒湖（Lake Akan）、冰岛玛花顿湖（Lake Mývatn）

日本阿寒湖附近小店内贩售的绿藻球

绿藻球的学名为 Aegagropila linnaei，在日语中球藻被称作マリモ（汉字：球藻，拼音：marimo）。在冰岛，球藻则被当地渔民称为 kúluskítur。（kúla = 球，skítur = 任何会被卷入渔网的水生植物）

日本阿寒湖绿藻球博物馆内的绿藻球

除了日本的阿寒湖，类似绿藻植物还被发现于冰岛、苏格兰、爱沙尼亚这些北半球高纬度的少数湖泊中，另外2014年在澳大利亚也发现了此类植物。细丝状的个体在某些情况下会形成绿色球状集合体，外观看起来就像是绿色的毛线球，大小从直径1公分到30公分都有。但是类似绿藻球这样的球状团生集合体目前只在日本、冰岛和爱沙尼亚被发现。

野外环境下的绿藻球

而最近日本的DNA研究发现了新的亚种，立山球藻（タテヤママリモ，Aegagropila linnaei sp.），但尚未正式发表，故学名尚未确定。参考网站

Marimo绿藻球比较容易饲养，另外对环境的要求很低，一般饲养绿藻球建议不要让绿藻球直接晒到太阳，因为绿藻球对光照的要求不是很高，光照太强容易长其他杂藻，另外由于绿藻球仅在高纬度湖泊被发现，可见绿藻球可以忍受低温的环境，因此如果夏天天气太热尽量将绿藻球饲养在阴凉的地方才好，冬天也完全无需为其加温。

绿藻球的生长形态

绿藻球主要有三种团生形态，主要由于河流底部的环境决定：

  1. 生长在岩石上的个体，通常生活在岩石的背光面。
  2. 在水中自由漂浮的丝状个体，通常会集合成一簇，在湖泊泥泞的底部形成一片绿色的绒毯状
  3. 球状集合体，丝状的绿藻个体从中心辐射状往外生长，形成大小不等的球体。
    野外湖底大小不一的绿藻球

绿藻球基本上生活在淡水中，但在淡水及盐水混合（盐度0.5-30‰）的水域，即半海水环境中也可发现。绿藻球的耐冷性及耐暗性非常的强，在冰箱中冷藏数个月也不会死。这是由于绿藻球生长的湖泊都位于高纬度地区，冬天的结冰期非常的长。但是绿藻球非常的怕热，耐热极限约为35℃。一般绿藻球是沉在湖底的，但随着光合作用的进行，氧气气泡附着在绿藻球上，也有可能使之浮起。球状体的生长速度约为一年5毫米。在日本阿寒湖，绿藻球个体直径约为20-30公分。

绿藻球的生长速度

温度：5-22℃培养5个月的结果表明，5℃生长最慢（1-1.5mm），22℃生长最快（23mm）。不要以为marimo耐寒，就会在冷水中生长更好。耐寒不等于喜寒。生命活动需要酶的催化，一定温度下才能保证酶的活性。Simon自己也养了几颗绿藻球，一直养在一个亚克力盒子里，发现和上面的研究结果一致，即绿藻球在大约20℃以上的环境下生长还是挺快的，注意不要光照太多就好，另外需要注意的是，绿藻球需要经常拿出来捏一捏，不然就会炸毛，每次捏完绿藻球，我就觉得它们瞬间变得很可爱。

炸毛的绿藻球

pH值： pH低于4.5和高于9的环境中，marimo基本无法正常生存。在pH值是7的中性水中marimo的生长速度是最快的（15-25mm）。阿寒湖イベシベツ川河口的pH值是7.7，相对于pH7，生长速度稍慢（10-15mm）。

营养物质： 最关键的是金属元素钙和镁。Marimo绿藻球在缺钙时，藻丝会很快变白枯死。而镁是构成叶绿素的重要元素，缺少镁的摄入，marimo也无法正常生存。这两种元素是marimo生存必不可少的元素。 其他的营养物质，钠，钾，磷，氮，硫，以及多种金属元素同样会对marimo的生长造成一定影响。
（部分数据来自阿寒湖mairmo调查研究会）

绿藻球长大之后里面是怎么样的呢？

球状个体的表面全都是绿的，所以绿藻球不论被翻到哪一面都能够行光合作用。但是相信养过绿藻球的朋友都会很好奇，绿藻球里面的部分长期见不到光，我们知道植物照不到光是会发黄发白的，那么绿藻球里面是怎么样的呢？

答案是绿藻球的内部也是绿的，因为那里充满了休眠中的叶绿体。如果球状体崩解，这些叶绿体就会苏醒，继续行光合作用。水流运动也会将覆盖在球状个体表面的砂石清走。有些球藻群落堆叠了二到三层的球状个体，所以必须要靠水流运动的翻滚，使每一个球状体都能够照到光线。

没有得到充分爱护和正确饲养的绿藻球炸毛、松散、褪色

绿藻球非常适合饲养在虾缸内，它们可以有效的吸收水中的氨和氮氧化物，也能提供一定的氧气。所以把绿藻球放在小缸中也是非常不错的选择，我自己就曾经用亚克力板粘了一个10x10cm的小缸，饲养了一条巧克力娃娃鱼和即可绿藻球，缸中的生态维持得非常好。当然你也可以把几只米虾和绿藻球养在一起，放在自己的办公桌上，一定会很好看的，还能够在累的时候看看它给自己放松放松。
