鱼缸的氮循环：如何建立硝化细菌系统

过滤 教程

要保持鱼缸水质的优质，最重要的就是建立好完整的氮循环系统，而硝化细菌 Nitrifying Bacteria则是该系统中最为重要的一环。很多水族爱好者会在网上看到出售硝化细菌粉剂或者硝化细菌水的商家，今天51热带鱼水族社区就来为大家好好介绍一下什么是硝化细菌，并且有没有必要购买这些所谓的“成熟”硝化细菌呢？

除了硝化细菌之外，其实鱼缸中还存在很多其他不可缺少的细菌，不少鱼友对硝化细菌的认识产生了一定的误解，有的人认为硝化细菌能够分解粪便；有的认为可以净化水质，中和水中的悬浮物，这些认识是不准确的，或者可以说是错误的，硝化细菌是生产者。

鱼缸内有机物的分解是靠其它分解者细菌完成的。在水生态循环系统中，若无其它异养性细菌存在，水中将到处充斥未被细菌分解的有机物，此种自我污染的水族环境一样使鱼儿无法生存其中。它们常被视为是水质自净作用的先锋部队，其重要性并不亚于硝化细菌。这类细菌普遍存在于各种不同环境，它们几乎无所不在，而繁殖速度相当惊人，大部份的异营性净水细菌一般只需二十几分钟即能增殖一倍。但是由于鱼缸内的设置很多时候我们投喂的鱼食和粪便无法被足够的分解者分解，因此在实际饲养中我们还是需要依靠物理过滤系统以及换水等措施来清除大部分的废物。

硝化细菌是是一类好氧性细菌的统称，主要包括亚硝酸菌（nitrosomonas）和硝酸菌（nitrobacter）。

分解细菌在分解有机质残渣的过程中会分解蛋白质，产生氨（ammonia），而氨是有毒的。此时上面提到的亚硝酸菌就出场了，它们将氨转化为亚硝酸盐，亚硝酸盐相比于氨来说毒性较小。

亚硝酸盐被硝酸菌转化为硝酸盐，少量低浓度的硝酸盐是对鱼虾无害的，但是长此以往水中的硝酸盐浓度过高仍对鱼虾致命，这时候我们就需要进行换水来降低鱼缸内硝酸盐的浓度。这也就是为什么定期定量地对鱼缸换水是如此重要，另外也可以在鱼缸中养殖一些水生植物或水草来降低水中的硝酸盐浓度（硝酸盐是植物生长必须的氮肥的优质来源，参考我的另外一篇文章 ► 一张图看懂鱼菜共生系统 Aquaponics System）。

缸内有益的细菌可以在缸内所有的表面上附着并进行生长繁殖，因此我们经常会建议使用表面积大的过滤材料来增加细菌可以附着的地方。很多商家会吹嘘用一些特殊材料制作的滤材有多好的效果，但其实生化过滤的材料并不是关键，一定体积内的表面积多少才是滤材好坏的关键。将一块用过的过滤棉放入一个新的过滤箱有助于在新的鱼缸中加速细菌菌落的建立。养鱼先养水，这边说的养水，其实就是养好水里这些看不见的细菌，如果能保证这些有益细菌的良性生长状态，那么就能在很大程度上保持水质的稳定和健康，从而可以让鱼儿不容易生病。

那么我们是否需要购买网上出售的硝化细菌胶囊或者液体呢？

答案是：一般而言不需要！鱼缸中氧含量和有机物多达到正常水平后，1周左右就可以建立起稳定的菌落。硝化细菌也不是药物，一旦稳定之后，只要环境不发生剧烈变化（如放入杀菌剂、UV等或开水倒入），就可以长期不断繁殖，完全无需添加。

另外这边说的目前市售的硝化细菌并不是假的，而是很难判断他们出售的硝化细菌里面含有有效的菌落活性和有效性，因此很多时候你就是买了一瓶水。由于硝化细菌培养技术难度较高，一般市面上出售的有效的硝化细菌如下图，另外有一些粉状的硝化细菌据说也有效果：

而事实上我们可以在开缸时做一些调整来加快有益菌的生长情况：

  1. 新鱼缸放水，加盐或者高锰酸钾进行消毒；
  2. 将之前消毒的水放干净后再把缸清洗一遍，加满水，放入滤材过滤棉，打开过滤、氧气泵，保持水温在25～28度左右；
  3. 约1～2天后在过滤棉上放置一些碎鱼虾肉（或者鱼饲料、鱼虾干也可以），为硝化细菌提供食物来源（如果你已经买了“硝化细菌”这个时候也可以往里面放一些）。在这期间保证鱼缸24小时开过滤，打氧气，滤材遮光；
  4. 观察水质变化，鱼缸可能在头几天变浑浊，之后变清，这时候可以放入闯缸鱼测试鱼缸水质；
  5. 闯缸鱼放入一星期后若仍不死则表示初步硝化系统建立完成，此时二级硝化系统尚未建立完全，仍要对鱼缸水质进行监控，大约2周后鱼缸水质稳定，可以开始正常养鱼；
  6. 之后只要正常维护，并定期进行换水就可以保证水质正常稳定。
    另外，如果身边有养鱼的朋友，那么去他们家接一盆他们清洗过滤棉后洗下来的“脏水”，倒入自己的鱼缸过滤系统中，则会将这个过程加速数倍！

最后我们来看一些数据，来帮助大家调整自己的鱼缸环境：

温度

硝化细菌的最佳生长温度在 77～86℉(25～30℃)之间。

  • 硝化细菌会在 32℉ (0℃) 下死亡。
  • 在 39℉ (4℃) 时不会发生活动，
  • 生长速率在 46~50℉ (7~10℃) 时下降75％
  • 长速率在 64℉ (18℃) 时下降低50％。
  • 硝化细菌会在 120℉ (49℃) 下死亡

硝化菌对低温的耐受性低于亚硝化单胞菌。在冷水系统中，必须小心监测亚硝酸盐的积累。

这也可以解释为什么用小缸养金鱼如果换水不勤的话金鱼很容易死亡，原因一般就是缺氧或者亚硝酸盐的累积。

pH值

  • 亚硝化单胞菌的最适pH范围为7.8～8.0。
  • 硝酸杆菌的最适pH范围为7.3～7.5

硝酸杆菌在海水水族箱典型的高pH水平下生长更缓慢，非洲Rift Lake Cichlids优选。可能存在初始高亚硝酸盐浓度。在pH水平低于7.0时，亚硝化单胞菌生长更慢，并且氨的增加可能变得明显。亚硝基单胞菌生长在pH6.5时被抑制。如果pH降至6.0或更低，则所有硝化被抑制。如果pH开始降至接近6.5，则必须小心监测氨。在该pH下，存在于水中的几乎所有的氨将处于轻微有毒的，离子化的NH 3 +状态。
