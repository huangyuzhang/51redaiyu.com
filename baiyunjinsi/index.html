白云金丝鱼（Tanichthys Albonubes）

原生鱼 小型鱼 图鉴

白云金丝鱼属鲤形目，鲤科唐鱼属。俗称：红尾鱼、白云金丝鱼、白云山鱼、邓鱼。英文名：White cloud mountain minnow 。原生地为中国广东省及海南省。

中文名：白云金丝鱼、唐鱼、邓鱼
学名：Tanichthys albonubes
英文名：White cloud mountain minnow
分类学：鲤形目鲤科唐鱼属
成鱼体长：3cm
适宜水温：14～25℃
适宜pH：6.0～8.5
濒危等级：中国国家Ⅱ级保护野生动物。

白云金丝成鱼可以长到3厘米左右，雄鱼各鳍较长，身体较瘦，颜色鲜艳，经常互相夸示。雌鱼鱼鳍较为短小，颜色暗淡，繁殖期的雌鱼腹部膨大。白云金丝鱼为杂食性鱼种，接受各种人工饲料以及活体饲料如冰冻血虫、丰年虾等，喂食活体饲料可以促进鱼只的发色和增加白云金丝的活力。

白云金丝适宜水温为14～25摄氏度，因此仅在冬季水温低于10摄氏度以下才需加热，以51热带鱼实际饲养经验来说若长期饲养在较热的水体中会导致缩短其寿命。

白云金丝如何繁殖

当发现白云金丝雄鱼开始频繁追逐雌鱼时，捞出1母2公到一个繁殖缸中，缸底要铺一些植物防止亲鱼吃卵。将繁殖缸放在温暖有阳光的地方。当发现雌鱼肚子变瘪、缸底有晶莹剔透的鱼卵时将亲鱼捞出。有条件的话可以用胶头滴管将鱼卵吸出，注意鱼卵较有粘性不要强行从附着物上取下来，否则会弄破鱼卵。若鱼卵粘在植物上可以连植物一同取出。条件允许的话将鱼卵置于打过2小时氧的等温蒸馏水中，这样可以大大减少鱼卵死亡的概率。将盛卵的器皿放在温暖有阳光处可以加快孵化速度。但注意温度不要超过26摄氏度。白云金丝鱼卵根据温度会在1～7天内孵化，刚孵出的幼鱼通体透明。3天后能看出眼睛等。在第五天将鱼苗放入有绿水的容器里，等到再大一些就可以喂食丰年虾无节幼虫了。
